# Intellij Recon Plugin 
IntelliJ plugin for Recon files

[![Build Status](https://app.travis-ci.com/swimos/intellij-recon.svg?branch=main)](https://app.travis-ci.com/swimos/intellij-recon)

### Running
1) `./gradlew :generateAll`
2) `./gradlew :runIde`

### Tests
1) `./gradlew :generateAll`
2) `./gradlew :test`
